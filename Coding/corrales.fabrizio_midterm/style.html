Styling Text

How to Style Text

The Styling text is very important in web design it can be used to add an attractive appearance to your website.Styling Text can also be used to add visual organization to your website allowing your users to navigate it more easily. This tutorial is on HTML style attribute, HTML background color, HTML text color, HTML fonts, and HTML text alignment.

The HTML Style Attribute

Setting the style of an HTML element, can be done with the style attribute. The HTML style attribute has the following syntax:

< tagname style=" property: value;">

The property is a CSS property. The value is a CSS value. You will learn more about CSS later in this tutorial.

HTML Background Color

The background-color property defines the background color for an HTML element.

This example sets the background color for a page to powderblue:

< body style="background-color:powderblue;">

< h1 > This is a heading < /h1 >

< p > This is a paragraph. < /p >

< /body >

HTML Text Color

The color property defines the text color for an HTML element:

< h1 style="color:blue;"> This is a heading < /h1 >

< p style="color:red;"> This is a paragraph. < /p >

HTML Fonts

The font-family property defines the font to be used for an HTML element:

< h1 style="font-family:verdana;"> This is a heading < /h1 >

< p style="font-family:courier;"> This is a paragraph. < /p >

HTML Text Alignment

The text-align property defines the horizontal text alignment for an HTML element:

< h1 style="text-align:center;"> Centered Heading < /h1 >

< p style="text-align:center;"> Centered paragraph. < /p >

In Conclusion, this is how to style text in web design.

Adding Links

By displaying links in your web page you can create different pages that show different information and connect them together or you can connect your web page to another page that you might want to direct the users on your webpage to. For instance you can use the tag :

<a href="http://www.google.com/">Google target=_blank</a>

In this tag I am making a link to the website google and this is a external link as which is a link that differs from my HTML webpages if you were to include an internal link you would change target=”_blank”to target=”_self”.This makes it not open up another tab.The “_blank” attribute creates a new tab for this website.This makes sure the user does not forget about your webpage.

 

To include a email you could use the tag <a href="mailto:someone@example.com?Subject=Hello%20again" target="_top">. You would put your email after “mailto:”

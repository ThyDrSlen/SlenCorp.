Glossary

  • Href = The href attribute specifies the URL of the page the link goes to.
  • Target = Specifies where to open the linked content
  • Title = Title of content
  • Src = the image source or location where the image can be found
  • Alt = an alternate tag allowing a nickname for a target

 

  • Height = sets the Url of the page the link goes to
  • Width  = sets the width of an element
  • Align  = aligns text or an image to the center of the page.
  • Border = Lines and color around text
  • Style  = define style information of the content on the web page

 

  • <html> = type of document and how to start the web page
  • <head> = The <head> element is a container for all the head elements.
  • <title> = The title tag used to give a name to a document seen on the tab’s website.
  • <style> = used to give different traits to tags in the body
  • <p> = paragraph tag to start paragraph format.
  • <br> = a line break for spacing content
  • <hr> = a header ruler used to divide content
  • <h1-h6> =The <h1> to <h6> tags are used to define HTML headings.
  • <ol> = an ordered list tag to create a list with numbers.
  • <ul> = a unordered list tag to create a list with bullets.
  • <em>=The italic tag to italicized text.(Used to show emphasis)
  • <strong> = darkens texts (makes text bold)
  • <i>=The italic tag to italicized text
  • <b>darkens texts (makes text bold)
  • <a>=The <a> tag defines a hyperlink, which is used to link from one page to another.
  • <img> = an image tag
  • <table> = starts a table format
  • <tr> = Table Row
  • <td> = Table Data
  • <th> = Table Header
  • <!--...--> the comment tag = The comment tag is used to insert comments in the source code
  • <li> = list item

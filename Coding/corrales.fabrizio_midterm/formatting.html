Formatting Text

Text Formatting is a simple way of allowing you to express your content in different ways.The main tags and what they do are as follows;<b> - Boldens text; <strong> - Same as <b> ;<i> - Italic text <em> - Same as Italicized text. These tags allow you to display content in different ways which makes easier convey content in the way you want to.

 

Formatting text is one of the easier things to do in HTML.All you need to do is write the open tag at the start of the content that you want formated then write the close tag at the end of the same content.Example to the left.

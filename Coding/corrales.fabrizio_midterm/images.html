Adding Images

Inserting images is a easy way to add appearance to your web page and also one of the simpler things to making a html web pages.

 

You can easily add images to web pages by using the tag.The tag is <img src="" alt="" height="" width="">. In the img src=”” you put the filename of the image. The alt=”” is where you could put an alternate name for the image file name if the file name is too long think of this as a nickname for the file name.The width is how wide the image is and is measured in pixels the height is how tall or short the image is and is also measured in pixels. You can also align an image by including the attribute align=”” in the tag you can put center,middle,left,right as the positions where you can put the image.
